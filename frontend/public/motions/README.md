# Character Animation Assets

Place these files here to enable the realistic runner:

- `character.glb` (rigged humanoid base model)
- `running.glb` (run animation clip)
- `jump.glb` (jump animation clip)
- `slide.glb` (slide/crouch/duck animation clip)

Notes:

- Use the same skeleton/rig for all files (Mixamo works well).
- Export in binary GLB format.
- If files are missing, the game automatically falls back to the procedural character.
